## 1. 수정 개요

YYYY-MM-DD 리뷰 회의에서 확정된 N개 수정 요청을 모두 반영했다.

| 구분 | Before | After |
|------|--------|-------|
| 교안 줄 수 | N,NNN줄 | N,NNN줄 (-NN.N%) |
| 소스 파일 수 | N개 | N개 |
| deprecated API | N건 (교안) + N건 (소스) | 0건 |
| 보안 이슈 | N건 | 0건 |

> 특이사항이 있으면 여기에 기술 (예: 목표 미달 사유, 사용자 판단에 의한 변경 등).
